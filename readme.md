# Dotfiles

## Structure

```shell
├─Application
│  ├─.glaze-wm    			A window manager for windows
│  ├─FlowLauncher			Quick file search & app launcher for Windows
│  ├─LX-music				Music App
│  └─Vivaldi				Browser
├─MISC						Miscellaneous Config
└─Terminal
    ├─CLI&TUI				CLI & TUI Utils
    │  ├─bat
    │  ├─bottom
    │  ├─fastfetch
    │  ├─gitui
    │  ├─scoop
    │  └─yazi
    ├─Shells				Shells' Config and Theme
    │  ├─nushell
    │  ├─oh-my-posh
    │  └─Powershell
    │      └─Scripts
    │          └─InstalledScriptInfos
    └─Terminal Emulator		Terminal Config
        ├─Wezterm
        └─Windows Terminal
```
